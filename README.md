Breast Cancer Prediction using Machine Learning

This project focuses on building a machine learning model to predict whether a tumor is benign or malignant based on diagnostic features. The goal is to assist in early detection of breast cancer using data-driven techniques.

The dataset used contains various features computed from digitized images of fine needle aspirates (FNA) of breast masses, such as radius, texture, perimeter, area, and smoothness.

Key Features
Data preprocessing and cleaning
Exploratory Data Analysis (EDA) with visualizations
Feature selection and scaling
Implementation of multiple ML algorithms (e.g., Logistic Regression, Decision Tree, Random Forest, SVM)
Model evaluation using accuracy, precision, recall, and confusion matrix
Comparison of model performances
🛠️ Tech Stack
Python
NumPy, Pandas
Matplotlib, Seaborn
Scikit-learn
Outcome

The models are trained and evaluated to achieve high accuracy in predicting breast cancer, helping demonstrate how machine learning can support medical diagnosis.
